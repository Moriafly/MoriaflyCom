# OpenMrw

## 开始使用

项目包发布在 JitPack 仓库，以下徽标显示了当前最新构建的版本：

[![](https://jitpack.io/v/xuncorp/openmrw.svg)](https://jitpack.io/#xuncorp/openmrw)

添加到项目：

```kotlin
// 将 Tag 替换为上面提到的徽标显示的最新版本号，如 0.1.0-dev03
implementation("com.github.xuncorp:openmrw:Tag")
```
