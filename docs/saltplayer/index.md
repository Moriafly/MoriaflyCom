## 跨平台统一体验

SPM (Salt Player Multi-platform) 是新一代跨平台音乐播放器，基于 Salt UI 和 Compose Multiplatform 构建，致力于在各平台提供一致且优质的本地音乐播放体验。

### 平台支持状态

| 平台 | 状态 | 说明 |
|:--|:--|:--|
| Android | ✅ 已发布 | 完整功能支持，各大应用商店可下载 |
| Windows | ✅ 已发布 | Microsoft Store 和 Steam 双渠道 |
| HarmonyOS | 🚧 开发中 | 原生鸿蒙应用开发中 |
| macOS | 🚧 开发中 | 即将推出 |
| Linux | 🚧 开发中 | 即将推出 |
