## Windows 专属特性

- **云母 / 亚克力效果** - Windows 11 原生视觉风格
- **桌面歌词** - 支持锁定、调整大小、改变颜色
- **系统媒体控制** - 支持 SMTC、任务栏缩略图
- **创意工坊** - 自定义主题、扩展功能
- **Steam 集成** - 丰富状态展示
